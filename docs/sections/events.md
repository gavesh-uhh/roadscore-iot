# Event Definitions

## Negative Events
- CRASH: Severe impact, sound, speed drop
- HARD_BRAKE: Sudden deceleration
- HARSH_ACCELERATION: Sudden acceleration
- SHARP_CORNER: Excessive roll/yaw
- POTHOLE: Strong vertical impact or vibration
- OVERSPEED: Exceeding speed limit
- UNDERSPEED: Driving too slow

## Positive Events
- SMOOTH_BRAKE: Gentle, controlled braking
- SMOOTH_ACCELERATION: Gentle, controlled acceleration

---

## Event Structure
- type: Event name
- alertType: Used for alert mapping
- severity: high/medium/low/positive
- penalty/bonus: Points deducted/added
